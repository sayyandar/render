Common Phishing Examples

Learn to identify different types of phishing attacks

Email Phishing

Example URL:

https://secure-paypal-login.verification-service.com

Red Flags:

  • Subdomain mimics legitimate service
  • Multiple hyphens and dots
  • Urgent action required
  • Misspelled company name

Banking Fraud

Example URL:

http://bankofamerrica-security-alert.com/verify?id=893745

Red Flags:

  • Misspelled bank name ("amerrica" instead of "america")
  • HTTP instead of HTTPS
  • Suspicious parameters in URL
  • Generic security alert theme

Social Media Scam

Example URL:

https://facebook-login-recovery.xyz/confirm?user=random&token=abc123

Red Flags:

  • Uses .xyz domain (less common for legitimate services)
  • Contains multiple query parameters
  • "Recovery" theme creates urgency
  • Unusual URL structure

Legitimate Example

Example URL:

https://www.paypal.com

Legitimate Indicators:

  • Correct spelling of company name
  • HTTPS protocol
  • Simple, clear URL structure
  • Official domain extension (.com)

Detection Tips

Our system detects phishing by analyzing URL patterns, character frequency, and structural anomalies. The examples above would be flagged by our AI model based on their suspicious characteristics.
